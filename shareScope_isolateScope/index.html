Isolate Scope:

Isolate Scope without data binding methoed. Won't be able to pass anything from controller to this directive.

Isolate Scope with one way data binding '@':

Use '@' to bind a local scope property to the value of a DOM attribute. The result is always a string.

Isolate Scope with two way data binding '=':

Use '=' to create a two-way data binding between a local scope property and the parent scope property.

Isolate Scope With Event '& '

Use '&' to execute an expression/function in the context of the parent scope.
